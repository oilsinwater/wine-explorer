# Development Workflow

## Local Development Setup

### Prerequisites

```bash
# Node.js (version 16+)
# npm or yarn package manager
```

### Initial Setup

```bash
# Clone repository
git clone <repository-url>
cd wine-explorer

# Install dependencies
npm install

# Start development server
npm run dev
```

### Development Commands

```bash
# Start all services
npm run dev

# Start frontend only
npm run dev

# Run tests
npm test
npm run test:unit
npm run test:e2e

# Build for production
npm run build
```
